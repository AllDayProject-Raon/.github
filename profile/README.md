## 소개

디지털 자격증명과 신원 기술을 실제 제품과 서비스에 안정적으로 연결하는 데 집중합니다.
사양에 머무르지 않고, 운영 가능한 구현과 상호운용 가능한 구조를 만드는 것을 목표로 합니다.

## 주요 분야

| 분야 | 설명 |
|:--|:--|
| 검증가능한 자격증명 | 발급, 제출, 검증 흐름 전반에 필요한 기술을 다룹니다. |
| 선택적 정보공개 | SD-JWT VC 기반의 프라이버시 중심 자격증명 교환을 지원합니다. |
| 모바일 자격증명 | mDoc 기반 시나리오와 기기 중심 사용 흐름을 구현합니다. |
| 상호운용성 | 다양한 표준과 포맷이 실제 환경에서 함께 동작하도록 맞춥니다. |

## 지원 표준

| 구분 | 항목 | 상태 |
|:--|:--|:--|
| 자격증명 포맷 | SD-JWT VC | ![지원](https://img.shields.io/badge/지원-완료-16a34a) |
| 자격증명 포맷 | W3C VC DM (JWT, JSON-LD) | ![지원](https://img.shields.io/badge/지원-완료-16a34a) |
| 자격증명 포맷 | mDoc Format | ![지원](https://img.shields.io/badge/지원-완료-16a34a) |

## 우리가 중요하게 보는 것

- 개방형 표준을 우선합니다.
- 실제 서비스에 투입할 수 있는 구현을 지향합니다.
- 보안성과 이동성을 기본 요구사항으로 봅니다.
- 실험에서 끝나지 않고 운영 가능한 구조를 만듭니다.
